What is JavaScript?

JavaScript is a programming language. It allows you to change the behavior, making web pages more interactive.


Variables

A variable in JavaScript is a data assigned naming convention. It's how we declare variables in Javascript. An example is var userInput = 1;.

    Number Numeric data-
